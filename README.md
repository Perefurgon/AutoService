# Информационная система "Автосервис"

## Описание проекта

Курсовая работа по дисциплине "Объектно-ориентированное программирование".

Консольное приложение для автоматизации работы автосервиса, включающее:
- Учёт клиентов и их автомобилей
- Управление заявками на обслуживание
- Справочник видов работ
- Авторизация с разграничением прав (Администратор/Оператор)
- Формирование отчётов (квитанции, выручка, статистика)

## Сборка в Visual Studio 2022

### Способ 1: Через CMake

1. Откройте Visual Studio 2022
2. Выберите "Открыть локальную папку" (Open a local folder)
3. Выберите папку AutoService
4. Visual Studio автоматически обнаружит CMakeLists.txt
5. Выберите конфигурацию x64-Debug или x64-Release
6. Сборка: Ctrl+Shift+B
7. Запуск: F5

### Способ 2: Создание проекта вручную

1. Создайте новый проект Console Application (C++)
2. Добавьте все .cpp файлы из папки src/
3. Добавьте все .h файлы из папки include/
4. В свойствах проекта добавьте include/ в "Дополнительные каталоги включения"
5. Установите стандарт C++17: Свойства → C/C++ → Язык → Стандарт языка C++ → ISO C++17

## Сборка в Linux (g++)

```bash
cd AutoService
make
./AutoService
```

## Реализованные принципы ООП

1. **Абстрагирование** - абстрактный класс Person
2. **Инкапсуляция** - приватные поля, публичные геттеры/сеттеры
3. **Наследование** - Client, Employee, User наследуют от Person
4. **Полиморфизм** - виртуальный метод getInfo()
